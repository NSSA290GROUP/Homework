1) run ServerGUI.java
2) run Chat.java
3) enter a username, your IP address (can find it on the server GUI when it pops up?)
4) select a protocol and click submit
5) go ahead and chat away!
6) When done, you can either close the chat via the red x or by selecting quit from the dropdown menu
7) If you want to have multiple chats open, that's also a possibility from the drop down menu 
	a) Open up a whole new chat 
	or 
	b) Join a chat already in progress
